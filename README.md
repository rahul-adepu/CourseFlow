# CourseFlow - Student Registration System

A simple student registration system built with React and Vite.

## Features

- **Course Types Management**: Create, list, update, and delete course types (e.g., Individual, Group, Special)
- **Courses Management**: Create, list, update, and delete courses (e.g., Hindi, English, Urdu)
- **Course Offerings**: Create, list, update, and delete course offerings by associating courses with course types
- **Student Registrations**: Register students for course offerings and view registrations with filtering capabilities

## Tech Stack

- React 18
- Vite
- Modern CSS

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```
